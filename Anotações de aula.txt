# Anotações - Bootcamp da Spread

## Git e Github

Git é um software de controle de versão (VCS), servindo para o versionamento de arquivos.
Github é o local ideal para guardar esses arquivos. É também uma espécie de rede social para programadores.

### Principais comandos do terminal

- echo [texto desejado]
Apresenta um texto no console

-cd [caminho desejado]
Navega para outra pasta

- ls/dir
Mostra os elementos presentes na pasta

- pwd
Mostra em qual path o usuário está no momento, na árvore de diretórios

- mkdir [nome da pasta desejada]
Criar uma nova pasta

- rm [nome do arquivo]
Apaga um arquivo

- rm -ref [nome da pasta]/del/rmdir
Apaga uma pasta

- sudo [comando]
Permite rodar comandos que necessitam de privilégio de super usuário

- touch [nome desejado do aquivo]
Cria ou atualiza um arquivo

- cat [nome do arquiv]
Captura o conteúdo de um arquivo e mostra na tela

- less [nome do arquivo]
Mostra o conteúdo de um arquivo no console

- history
Mostra o histórico de comandos

- grep [texto][arquivo desejado]
Filtra o conteúdo de um arquivo através do texto desejado e mostra na tela o resultado

- cp [nome do arquivo][nome do novo arquivo]
Copia o conteúdo de um arquivo para um outro arquivo

-mv [nome do arquivo][nome do novo arquivo]
Move o conteúdo de um arquivo para outro, apagando o anterior

- clear/cls 
Limpa os textos que estão aparecendo no console

- exit
Fecha a sessão atual
